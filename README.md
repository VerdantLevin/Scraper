Clone repo:
```commandline
git clone https://github.com/VerdantLevin/Scraper.git
```
Move to directory:
```commandline
cd Scraper/
```
Initialize venv:
```commandline
python -m venv .venv
```
Run venv:
```commandline
source .venv/bin/activate
```
or
```commandline
.venv/Scripts/Activate.ps1  
```
Install dependencies:
```commandline
pip install -r requiremwnts.txt
```
